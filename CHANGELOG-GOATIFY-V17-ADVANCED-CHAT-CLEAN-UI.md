# Goatify V17 — Advanced Chat Clean UI

## Objetivo
Rediseñar la interfaz del Chat Avanzado para que se sienta más limpia, moderna y cercana a ChatGPT/Gemini, sin quitar funciones ni botones esenciales.

## Cambios aplicados
- Se eliminó el panel lateral fijo dentro del área de conversación.
- Las herramientas pasan a un modal limpio llamado “Herramientas del Chat”.
- Se mantiene PlanCreditBadge, pero sin invadir la conversación.
- El chat queda centrado con ancho máximo, lectura más cómoda y más aire visual.
- Los mensajes tienen burbujas más sobrias, limpias y modernas.
- El input inferior queda estilo barra moderna tipo ChatGPT/Gemini.
- Se mantienen botones de:
  - Nuevo chat
  - Asignar proyecto
  - Borrar chat
  - Compartir / analizar pantalla
  - Pantalla completa
  - Manual
  - Adjuntar archivo local
  - Adjuntar desde Drive
  - Enviar
  - Voz
  - Video
- Se mantiene contexto de proyecto, calendario, Drive y memoria reciente, pero en chips discretos.
