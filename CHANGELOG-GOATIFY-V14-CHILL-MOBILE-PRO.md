# Goatify V14 — Chill Mobile Pro

Versión enfocada exclusivamente en mejorar Goatify Chill sin quitar módulos, botones ni tocar el resto del ecosistema.

## Mejoras clave

- Controles táctiles reutilizables para juegos (`MobileGameControls`).
- Goat Racer mejorado para celular:
  - botones táctiles izquierda/derecha;
  - nitro con press/hold;
  - modo giroscopio para curvar inclinando el teléfono cuando el navegador lo permite;
  - HUD de velocidad dentro del canvas;
  - fondo y pista con más profundidad visual.
- Goat Invaders mejorado para celular:
  - botones táctiles;
  - movimiento mantenido por press/hold;
  - botón de disparo.
- Pacman mejorado para celular:
  - swipe en el canvas;
  - cruceta táctil.
- 2048 mejorado para celular:
  - swipe en el tablero;
  - cruceta táctil alternativa.
- Neon Snake reforzado:
  - swipe existente conservado;
  - cruceta táctil alternativa.
- Dino Run reforzado:
  - taps existentes conservados;
  - botones táctiles de salto/agacharse.
- Flappy Goat reforzado:
  - pointer/touch más confiable;
  - área táctil optimizada.
- Goatify Chill dashboard:
  - tarjeta de torneo diario;
  - llamada visual a Goat Racer;
  - guía “Mobile Ready”.

## No se modificó

- Cloud Run.
- Login.
- Mail.
- POS.
- Chat avanzado.
- Social Media Studio.
- Presentaciones.
- Drive.
- Política Pro/Premium.
- Navegación principal.
